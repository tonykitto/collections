Object Entry Form

Information about objects entering the museum are recorded on paper using a standard Collections Trust Object Entry Form. The procedure, which closely follows Collections Trust's Documentation guidelines, is recorded in Towneley's Documentation Procedure Manual.

In the event of an object being accessioned, the Modes object entry will include the object entry form number. For more information on the relation between the museum's paper records, Modes and public versions of the Collection Management System - see Documentation System.
